# Media Guide

This folder is for screenshots and optional video previews used by the main README.

## Recommended files

- `gameplay-main.png`: home screen or mid-game overview
- `gameplay-chain.png`: chain explosion highlight
- `settings-overlay.png`: settings or tutorial overlay
- `gameplay.mp4`: optional local gameplay video
- YouTube demo: [https://www.youtube.com/watch?v=Ku6gNe-UeJE](https://www.youtube.com/watch?v=Ku6gNe-UeJE)

## Current asset layout

- `asset/img/`: interface and background images
- `asset/aud/`: background music tracks
- `asset/gameplay/`: gameplay-related media and demo video

## Notes

- Screenshots are enough for a professional README.
- Add video only if you want an extra showcase layer.
